**Environment**
|  |  |
|--|--|
| Rails version | 7.0.4.2  |
| Ruby version | 2.7.6  |
| Database | sqlite3  |
| Testing Framework | RSpec  |

**Clone The Repository**
```
git clone https://github.com/ripan/projecttalk.git
```

**Running BE API** 
```
- cd projecttalk
- rails db:create db:migrate db:seed
- rails s
```

**Running RSpec** 
```
- cd projecttalk
- RAILS_ENV=test rspec spec
```
